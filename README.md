# Character Forge

Character Forge is a single-page anime character themed web app built as a standalone HTML file.

## Preview

The app lives entirely in `index.html`, making it easy to open locally or deploy as a static page.

## Run locally

Open `index.html` directly in a browser.

## Project structure

- `index.html` - main application file, styles, and client-side behavior
- `README.md` - project overview and usage notes

## Deployment

Because this project is static, it can be hosted on platforms such as GitHub Pages, Netlify, or Vercel without any build step.
